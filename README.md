# LProg-phase-extra
GIT repository for the extra modifications to the x++ language, independent from the LProg course.

To build the executable, run `makeit.sh`. To run all tests, run `run_tests.sh`.

The additions made to LProg-phase2 are the following:
- parallel assignments - TODO
- n++
- ++n
- n--
- --n
- sumall
- try else - TODO

This repo will also include the SasyLF preservation theorem proof for labeled records, instead of only pairs.
